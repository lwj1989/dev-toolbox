## ğŸ“¦ Local Development

### Requirements

- Node.js 18+
- npm 9+

### Clone Repository

```bash
git clone https://github.com/your-username/dev-toolbox.git
cd dev-toolbox
```

### Install Dependencies

```bash
npm install
```

### Development Mode

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build for Production

```bash
# Build the application
npm run build

# Preview the build
npm run preview
```
